# 📱 USER BEHAVIOR INTELLIGENCE SYSTEM FOR MOBILE ANALYTICS

---

## 📌 OVERVIEW

This project analyzes mobile user behavior to understand screen time, app usage, battery consumption, and engagement patterns using **Python, SQL, and Power BI**.

---

## 🛠️ TOOLS USED

* Python (Pandas)
* MySQL
* Power BI

---

## 📁 PROJECT STRUCTURE

```
data/
python/
powerbi/
sql/
```

---

## ▶️ HOW TO RUN

1. Download dataset from `data/`
2. Import `cleaned_user_behavior.csv` into MySQL
3. Open `sql/queries.sql`
4. Run queries in MySQL Workbench
5. Open Power BI file (`.pbix`) to view dashboard

---

## 📊 KEY INSIGHTS

* Screen time directly impacts battery usage
* Higher app usage leads to increased data consumption
* Users can be segmented into Low, Medium, High usage groups
* Engagement varies across user segments

---

## 💡 RECOMMENDATIONS

* Optimize battery for heavy users
* Introduce screen time alerts
* Provide data-saving features
* Personalize features based on user behavior
